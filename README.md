## Usage

#### Clone project boilerplate

```
$ git clone git@github.com:alchaplinsky/module.git
$ git remote rm origin
$ git remote add origin [YOUR_REPO]
```

#### Build code 
```
yarn build
```

#### Run tests

```
yarn test
```
